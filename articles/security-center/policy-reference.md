# <a name="azure-policy-built-in-definitions-for-microsoft-defender-for-cloud"></a>Integrierte Azure Policy-Definitionen für Microsoft Defender für Cloud

[!INCLUDE [Banner for top of topics](./includes/banner.md)]

Diese Seite enthält einen Index der integrierten [Azure Policy](../governance/policy/overview.md)-Richtliniendefinitionen für Microsoft Defender für Cloud. Die folgenden Gruppierungen von Richtliniendefinitionen sind verfügbar:

- In der Gruppe [Initiativen](#microsoft-defender-for-cloud-initiatives) sind die Azure Policy-Initiativendefinitionen in der Kategorie „Defender für Cloud“ aufgeführt.
- In der Gruppe [Standardinitiative](#defender-for-clouds-default-initiative-azure-security-benchmark) sind alle Azure Policy-Definitionen aufgeführt, die Teil der Standardinitiative [Azure-Sicherheitsvergleichstest](/security/benchmark/azure/introduction) von Defender für Cloud sind. Diese von Microsoft erstellte, weit verbreitete Benchmark basiert auf Kontrollen des [Center for Internet Security (CIS)](https://www.cisecurity.org/benchmark/azure/) und des [National Institute of Standards and Technology (NIST)](https://www.nist.gov/) und konzentriert sich auf cloudzentrierte Sicherheit.
- In der Gruppe [Kategorie](#microsoft-defender-for-cloud-category) sind alle Azure Policy-Definitionen in der Kategorie „Defender für Cloud“ aufgeführt.

Weitere Informationen zu Sicherheitsrichtlinien finden Sie unter [Arbeiten mit Sicherheitsrichtlinien](./tutorial-security-policy.md). Weitere Azure Policy-Integrationen für andere Dienste finden Sie unter [Integrierte Azure Policy-Richtliniendefinitionen](../governance/policy/samples/built-in-policies.md).

Die Namen der einzelnen integrierten Richtliniendefinitionen sind Links zur entsprechenden Richtliniendefinition im Azure-Portal. Verwenden Sie den Link in der Spalte **Version**, um die Quelle im [Azure Policy-GitHub-Repository](https://github.com/Azure/azure-policy) anzuzeigen.

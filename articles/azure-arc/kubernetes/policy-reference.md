# <a name="azure-policy-built-in-definitions-for-azure-arc-enabled-kubernetes"></a>Integrierte Azure Policy-Richtliniendefinitionen für Kubernetes mit Azure Arc-Unterstützung

Diese Seite enthält einen Index der integrierten [Azure Policy](../../governance/policy/overview.md)-Richtliniendefinitionen für Kubernetes mit Azure Arc-Unterstützung. Weitere Azure Policy-Integrationen für andere Dienste finden Sie unter [Integrierte Azure Policy-Richtliniendefinitionen](../../governance/policy/samples/built-in-policies.md).

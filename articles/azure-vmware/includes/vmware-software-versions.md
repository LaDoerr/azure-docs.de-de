| vSAN                  |    6.7 U3l   |
| HCX                   |    4.1       |
| NSX-T <br />**HINWEIS:** NSX-T ist die einzige unterstützte Version von NSX.               |      [[!INCLUDE [nsxt-version](nsxt-version.md)]](https://docs.vmware.com/en/VMware-NSX-T-Data-Center/3.1/rn/VMware-NSX-T-Data-Center-312-Release-Notes.html)     |

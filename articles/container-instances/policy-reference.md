# <a name="azure-policy-built-in-definitions-for-azure-container-instances"></a>Integrierte Azure Policy-Definitionen für Azure Container Instances

Auf dieser Seite finden Sie einen Index der in [Azure Policy](../governance/policy/overview.md) integrierten Richtliniendefinitionen für Azure Container Instances. Weitere Azure Policy-Integrationen für andere Dienste finden Sie unter [Integrierte Azure Policy-Richtliniendefinitionen](../governance/policy/samples/built-in-policies.md).

Die Namen der einzelnen integrierten Richtliniendefinitionen sind Links zur entsprechenden Richtliniendefinition im Azure-Portal. Verwenden Sie den Link in der Spalte **Version**, um die Quelle im [Azure Policy-GitHub-Repository](https://github.com/Azure/azure-policy) anzuzeigen.

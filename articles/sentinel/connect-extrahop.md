# <a name="connect-extrahop-revealx"></a>Herstellen einer Verbindung mit ExtraHop Reveal(x)

Der ExtraHop Reveal(x)-Datenconnector ermöglicht es Ihnen, Ihr ExtraHop Reveal(x)-System auf einfache Weise mit Azure Sentinel zu verbinden, um Dashboards anzuzeigen, benutzerdefinierte Warnungen zu erstellen und die Untersuchung von Daten zu verbessern. Durch diese Integration können Sie einen besseren Einblick in das Netzwerk Ihrer Organisation gewinnen und die Möglichkeiten für Sicherheitsvorgänge verbessern.

[!INCLUDE [reference-to-feature-availability](includes/reference-to-feature-availability.md)]

## <a name="prerequisites"></a>Voraussetzungen

- Lese- und Schreibberechtigungen für den Azure Sentinel-Arbeitsbereich.

- ExtraHop Discover- oder Command-Appliance mit Firmwareversion 7.8 oder höher mit einem Benutzerkonto, das über unbegrenzte Berechtigungen (Administratorberechtigungen) verfügt.

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Verbinden von Azure Sentinel mit ExtraHop Reveal(x)

1. Navigieren Sie im Azure-Portal zu **Azure Sentinel** > **Datenconnectors**, und wählen Sie dann den Connector **ExtraHop Reveal(x)** aus.

2. Wählen Sie **Connectorseite öffnen** aus.

3. Befolgen Sie die Anweisungen auf der Seite **ExtraHop Reveal(x)**.

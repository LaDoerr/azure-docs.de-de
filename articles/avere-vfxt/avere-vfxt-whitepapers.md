# <a name="azure-vfxt-for-azure-whitepapers-and-other-resources"></a>Azure vFXT für Azure-Whitepaper und andere Ressourcen

Dieser Artikel umfasst Fallstudien, Technologiewhitepaper, Infografiken und andere Ressourcen, die Ihr Verständnis dafür verbessern können, wie Avere vFXT for Azure in Ihren Workflow passt.

## <a name="deploy-agisoft-photoscan-on-azure"></a>Bereitstellen von Agisoft PhotoScan auf Azure

**Link:** [Bereitstellen von Agisoft PhotoScan in Azure mit Avere vFXT for Azure oder BeeGFS](https://azure.microsoft.com/mediahandler/files/resourcefiles/deploy-agisoft-photoscan-on-azure-with-azere-vfxt-for-azure-or-beegfs/AgiSoft%20PhotoScan%20on%20Azure%20using%20Avere%20vFXT%20or%20BeeGFS.pdf)

Dieses Handbuch bietet eine Schrittanleitung zum Installieren der Agisoft PhotoScan-Fotogrammetriesoftware mit Unterstützung durch wahlweise das Avere vFXT-Speicherdateisystem oder das parallele BeeGFS-Dateisystem. Der Autor, Paulo Marques da Costa von AzureCAT, demonstriert das Einrichten von PhotoScan auf virtuellen Azure-Computern (VMs). Hochleistungsspeicher verringert die Verarbeitungszeit, und die Ergebnisse seiner Benchmarktests sind enthalten. Diese Umgebung kann ohne Kompromisse bei der Leistung nach Bedarf zentral hoch- und herunterskaliert werden und unterstützt Terabytes von Speicher.

#### <a name="c-nuget-package"></a>NuGet-Paket für C#

Das .NET Speech SDK kann mit dem folgenden `dotnet add`-Befehl über die **.NET Core-CLI** installiert werden.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

Das .NET Speech SDK kann mit dem folgenden `Install-Package`-Befehl über den **Paket-Manager** installiert werden.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```

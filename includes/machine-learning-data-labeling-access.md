Jeder Benutzer, der Zugriff auf Ihren Arbeitsbereich hat, kann Daten in Ihrem Projekt beschriften.  Sie können auch die Berechtigungen für Ihre Beschriftungsersteller anpassen, sodass sie zwar auf die Beschriftung, aber nicht auf andere Teile des Arbeitsbereichs oder das Beschriftungsprojekt zugreifen können.  Weitere Informationen finden Sie unter [Verwalten des Zugriffs auf einen Azure Machine Learning-Arbeitsbereich](../articles/machine-learning/how-to-assign-roles.md). Dort erfahren Sie auch, wie Sie die [benutzerdefinierte Rolle „Beschriftungsersteller“](../articles/machine-learning/how-to-assign-roles.md#labeler) erstellen.

> [!NOTE]
> Azure Machine Learning-Endpunkte (Vorschau) bieten ein verbessertes und vereinfachtes Bereitstellungserlebnis. Endpunkte unterstützen Szenarien mit Echtzeit- und Batchrückschluss. Endpunkte verfügen über eine einheitliche Schnittstelle zum übergreifenden Aufrufen und Verwalten von Modellbereitstellungen für alle Computetypen. Siehe [Was sind Azure Machine Learning-Endpunkte (Vorschau)?](../articles/machine-learning/concept-endpoints.md).

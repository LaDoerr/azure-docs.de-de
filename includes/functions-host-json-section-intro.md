In diesem Abschnitt werden die für diese Bindung verfügbaren Konfigurationseinstellungen in Version 2.x und höher beschrieben. Einstellungen in der Datei „host.json“ gelten für alle Funktionen in einer Funktions-App-Instanz. Die nachfolgende Beispieldatei „host.json“ enthält nur die Einstellungen für Version 2.x und höhere Versionen für diese Bindung. Weitere Informationen über die Konfigurationseinstellungen für Funktions-Apps in Version 2.x und höher finden Sie unter [host.json-Referenz für Azure Functions](../articles/azure-functions/functions-host-json.md).
